Frequently Asked Questions

Following questions were asked to answer for bonus marks. Hence, an explanation has been given to each question.

Grid mainly works based on layout while Flexbox works with the content. Grid is used for large-scale layout where the layout is not linear whereas Flexbox is used for small-scale layout.

Both are CSS frameworks but Bootstrap comes with a set of pre-styled responsive, mobile-first components that has a defined set of parameters while Tailwind CSS uses a set ofutility classes and allows more flexibilty and uniqueness. Bootstrap is quite famous and still used heavily whereas Tailwind CSS is new and slowly reaching up.

CSS box model is a container that has various properties: margin, borders, padding and the content itself. An element is rendered as a rectangle in the browser and these properties can be manipulated to give the element its customized layout.

Semantic tag is atype of HTML tag which has a meaning to it, giving information about the type of content that goes into it. For example article, header, footer or section.
